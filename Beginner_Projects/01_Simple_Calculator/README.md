#  Simple Calculator

##  Overview

The **Simple Calculator** is a beginner-friendly C programming project that performs basic arithmetic operations through a menu-driven interface.

This project helps beginners understand user input, decision-making, and arithmetic operations while following good programming practices.

---

#  Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🟰 Modulus
- 🚪 Exit Option
- 📋 Menu-Driven Interface
- ⚠️ Division by Zero Handling

#  How to Run

### Compile

```bash
gcc calculator.c -o calculator
```

### Run

#### Linux / WSL

```bash
./calculator
```

#### Windows

```bash
calculator.exe
```
